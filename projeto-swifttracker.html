Sobre o Projeto

Desenvolvi uma aplicação Full-Stack robusta para monitoramento e previsão da cotação do Dólar (USD/BRL) em tempo real. O sistema utiliza uma arquitetura desacoplada para garantir performance e escalabilidade.

O diferencial do projeto é seu "Core" de inteligência, que analisa 41 indicadores macroeconômicos (incluindo Commodities, Juros Americanos/Treasuries, Índices Globais e Criptomoedas) para projetar tendências de curto prazo.

Principais Funcionalidades:

  • Monitoramento em Tempo Real: Integração com AwesomeAPI para cotação live (atualização a cada 5s).
  • Algoritmo Ponderado: Backend em Python que coleta e processa 41 variáveis de mercado via Yahoo Finance para gerar um "Alvo IA".
  • Histórico de Assertividade: Sistema de log automático no Supabase que compara previsões passadas com a realidade, gerando uma lista de assertividade auditável.
  • Dashboard Interativo: Interface moderna em Dark Mode com gráficos e indicadores visuais de tendência.

Tecnologias Utilizadas:

  • Frontend: Next.js, TypeScript, Tailwind CSS.
  • Backend: Python, FastAPI, Pandas, yfinance.
  • Infraestrutura: Render (Hospedagem de Microsserviços), Vercel (Frontend).
  • Banco de Dados: Supabase (PostgreSQL) para logs históricos e análise de performance.
