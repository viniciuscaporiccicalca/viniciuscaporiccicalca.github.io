ServiceApp: Google Play

Sobre o Projeto

Desenvolvi e publiquei o Versa ServiceApp, uma solução SaaS completa para prestadores de serviços, utilizando uma arquitetura híbrida moderna que une a performance da Web com recursos nativos Mobile.

O projeto foi construído do zero, desde a concepção do Banco de Dados até o deploy na Play Store, resolvendo problemas reais de gestão como Ordens de Serviço, Controle Financeiro e Estoque.

Destaques Técnicos & Desafios Superados:

  • Arquitetura Híbrida: Utilizei Next.js com Capacitor para criar uma base de código única que roda tanto na Web (Vercel) quanto no Android (Nativo), otimizando o tempo de desenvolvimento.
  • Backend & Segurança: Implementação de Supabase (PostgreSQL) com Row Level Security (RLS) para garantir isolamento total de dados entre usuários (Multi-tenant) e Auth via Google OAuth com Deep Linking nativo.
  • Monetização: Integração complexa com RevenueCat e Google Play Billing para gestão de assinaturas recorrentes e notificações em tempo real via Google Cloud Pub/Sub.
  • Integrações: Emissão de NF-e via API REST e geração de PDFs dinâmicos no frontend.

Tech Stack:

  • Frontend: React, Next.js, Tailwind CSS, TypeScript.
  • Mobile: Capacitor, Android Studio.
  • Backend/BaaS: Supabase (Auth, DB, Storage, Edge Functions).
  • Infra: Vercel, Google Cloud Platform.
  • APIs: RevenueCat, Focus NFe.

Disponível na Play Store.
